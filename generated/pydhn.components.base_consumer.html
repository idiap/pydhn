Class for base Consumer component. It has no operational parameters, only a mass flow setpoint is specified.

Parameters:

  • control_type (str, optional) ‚Äì Control type to define how mass flow is imposed. The default is energy.

  • diameter (float, optional) ‚Äì Diameter (m). Unused. The default is 0.02.

  • mass_flow_min (float, optional) ‚Äì Minimum mass flow (kg/s) allowed. The default is 1e-06.

  • heat_demand (float, optional) ‚Äì Heat demand (Wh) used to compute the hydraulic setpoint in case the ‚Äòenergy‚Äô control type is used. The default is 5000.0.

  • design_delta_t (float, optional) ‚Äì Design temperature difference (K) of the heat exchanger used to compute the hydraulic setpoint in case the ‚Äòenergy‚Äô control type is used. The default is -30.0.

  • setpoint_type_hx (str, optional) ‚Äì Type of thermal setpoint to use. The default is delta_t.

  • setpoint_type_hx_rev (str, optional) ‚Äì Type of thermal setpoint to use in case of reverse flow. The default is delta_t.

  • setpoint_value_hx (float, optional) ‚Äì Value of the thermal setpoint. The default is -30.0.

  • setpoint_value_hx_rev (float, optional) ‚Äì Value of the thermal setpoint in case of reverse flow. The default is 0.0.

  • power_max_hx (float, optional) ‚Äì Maximum power (W) that the consumer can exchange. The default is nan.
